Rainbow

A set of colorful CSS3 buttons

Every button should have the base rainbow button class .rb. All the other classes add colors or presentation styles and are meant to be used together with the base. To keep the examples uncluttered, only the additional classes are listed.

By default the buttons do not have a fixed width, so they will expand according to their text. Feel free to set one in your own styles if you need to (as these examples have).

Common class names that are sometimes injected by javascript libraries do not use the rainbow prefix (rb-): selected, disabled, small.

When grouping buttons make sure you get rid of the default space between inline-block elements using one of these methods.
